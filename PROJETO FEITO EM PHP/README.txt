1- PARA RODAR O CODIGO É NECESSARIO TER XAMPP INTALADO
2- APÓS INSTALAR O XAMPP ABRIR ELE E INICIAR O APACHE E MYSQL
3- COM O MYSQL INICIADO CLICAR NA OPÇÃO ADMIN QUE IRA LHE REDIRECIONAR PARA A PAGINA DO PHPMYADMIN
4- NA PAGINA DO PHPMYADMIN VOCE DEVE IMPORTAR O BANCO DE DADOS "tela_login.sql"
5- APOS ISSO VOCE DEVE PEGAR A PASTA INTEIRA DO PROJETO "logintot" E COLOCAR DENTRO DA PASTA htdocs QUE FICA DENTRO DOS ARQUIVOS DE INTALAÇÃO DO XAMP:"xampp/htdocs/"
6- PARA FINALIZAR ABRA O NAVEGADOR E DIGITE: localhost/logintot QUE ABRIRA O PROJETO JA SENDO EXECUTADO
